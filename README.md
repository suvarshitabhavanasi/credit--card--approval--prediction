# Credit Card Approval Prediction using Machine Learning

## Project Overview
This project predicts whether a credit card application should be approved or rejected using Machine Learning algorithms.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- Google Colab
- GitHub

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

## Dataset
- application_record.csv
- credit_record.csv

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Feature Encoding
5. Model Training
6. Model Evaluation
7. Model Saving
8. Flask Web Application
